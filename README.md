# CutMonitor

Печатная плата контроля целостности сигнальной цепи и давления в пневматической системе

Функционал изделия:
- контроль целостности сигнальной цепи;
- управление четырьмя двигателями постоянного тока;
- измерение давления с четырех аналоговых датчиков;
- анализ сигнала с четырех оптопар;
- обработка сигналов микроконтроллером stm32f103rbt6;
- дистанционное управление и передача данных по wi-fi.
